# Infinity AI Lead Engine

AI-виджет для агентств недвижимости.

## Что умеет

- Автоматически открывается через 3 секунды.
- Спрашивает клиента, чем помочь:
  - купить недвижимость;
  - продать недвижимость;
  - арендовать;
  - сдать;
  - другой вопрос.
- Для покупки ищет объекты через `property-search`.
- Не показывает карточки в чате, а сообщает количество найденных объектов.
- Собирает контакт и важные пожелания.
- Сохраняет лид в Supabase через `save-lead`.
- Отправляет агенту Telegram-уведомление на сербском языке.
- Для WhatsApp и Telegram добавляет кнопки быстрого перехода.
